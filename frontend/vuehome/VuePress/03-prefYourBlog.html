# 0. 阅读前置

# 1. 主页配置

参考: 官方文档-Homepage

之前我们已经在 docs 目录下创建了 README.md 文件，现在只需要用下面内容直接替代现有 README.md 的内容即可：

---
home: true
heroImage: http://img.nodreame.cn/2020-3-23-21-27-49
heroText: Nodreame技术进阶
tagline: 蛾儿雪柳黄金缕，未若柳絮因风起
actionText: Get Started →
actionLink: /frontend/vuehome/VuePress/
features:
- title: Web Develop
  details: 别问，问就是全干
- title: 数据结构 & 算法
  details: 目标是 Leetcode 周赛写完四道 ToT
- title: 我是谁
  details: 一位误入码路的 Pokemon Master
footer: MIT Licensed | Copyright © 2020 Nodreame
---

这是由于默认的主题自带了 homepage 的布局，而我们通过 YAML 格式的 home: true 来启动了 homepage 样式.其他参数可以自行对比页面进行调整。效果如下所示:

# 2) 配置导航栏

参考：官方文档-Navbar

假设我的目标效果是：大前端(前端基础、Vue系列)、数据结构&算法（树、动态规划）、Github【外部链接】，那么基于 VuePress 的特性 Convention is better than configuration约定大于配置) 首先我就要构建好文件目录, 这样之后点击导航栏才能到达目标内容。构建目录树如下：



 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
├─ .vuepress
│  ├─ public        // 存储静态资源（用图床替代大部分静态资源,只存放 logo.png）
│  ├─ config.js     // 网站配置文件
├─ algorithms       // "数据结构&算法"文件夹
│  ├─ dp            // "动态规划"文件夹
│  │  ├─  README.md
│  ├─ tree          // "树"文件夹
│  │  ├─  README.md
├─ frontend         // "大前端"文件夹
│  ├─ basehome      // "前端基础"文件夹
│  │  ├─  README.md
│  ├─ vuehome
│  │  ├─  Vue       // "Vue"文件夹
│  │  │  ├─  README.md
│  │  ├─  VuePress  // "VuePress"文件夹
│  │  │  ├─  README.md
│  │  ├─  README.md
└─ README.md // 源码目录

每个文件夹下有一个 README.md用来展示内容, 接下来根据官方文档的指导在 config.js 中完成 navbar 的配置：




 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 



module.exports = {
  base: "/",
  title: "Nodreame技术进阶",
  themeConfig: {
    nav: [
      {
        text: '大前端',
        items: [
          { text: '前端基础', link: '/frontend/basehome/' },
          { text: 'Vue', link: '/frontend/vuehome/Vue/' },
          { text: 'VuePress', link: '/frontend/vuehome/VuePress/' }
        ]
      },
      {
        text: '数据结构 & 算法',
        items: [
          { text: '树', link: '/algorithms/tree/'},
          { text: '动态规划', link: '/algorithms/dp/'}
        ],
      },
      { text: 'Github', link: 'https://github.com/Nodreame' }
    ],
  }
}

navbar 的配置比较简单，完成上面的配置之后效果如下(点击导航栏中"VuePress", 显示 vuepress 文件夹下 README.md 文件内容)，效果如下所示：
