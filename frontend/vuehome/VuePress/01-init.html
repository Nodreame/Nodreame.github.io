# (一) 把 Hello 写到你的 GithubPage

目标效果: 实现一行命令将博客部署到 Github Page(无需购买服务器和部署后台).

# 一、创建博客工程

# 1.1 本地环境准备

  • 本地安装好 Node.js（版本>=8.0）, 推荐安装 yarn.
  • 具备可运行命令的终端
    • Window: cmder、Git Bash
    • Mac: Terminal、iTerm
    • Linux: 默认终端

# 1.2 用 VuePress 创建博客工程

# 创建博客工程
mkdir blog
cd blog
yarn add -D vuepress  # 或者 npm i -D vuepress(如果下载太慢可换用 cnpm)
mkdir docs
echo 'Hello, my first blog!' > docs/README.md  # 编写第一篇文章（注意符号）

# 二、部署到 Github Page

# 2.1 创建 Github Page 工程

Tip: 项目名必须符合标准, 项目名应为：Github用户名.github.io, 如下所示.

# 2.2 本地编写打包发布脚本

在 blog 项目根目录创建 build.sh，编写内容如下：

#!/usr/bin/env sh

# abort on errors
set -e

# build
yarn docs:build # 或者 npm run docs:build

# nav into the build out dir
cd docs/.vuepress/dist

# git: 每次项目重新打包都需要重新初始化 Git 并强制推送
git init
git add .
git commit -m 'deploy'
# 如果使用 SSH 方式需要修改链接
git remote add origin https://github.com/Nodreame/Nodreame.github.io.git
git push -f origin master

# 2.3 脚本打包并发布生成网站

  • 脚本权限设置：chmod +x build.sh
  • 运行脚本 ./build.sh, 开始部署

  • 查看 Github 上 blog 项目的文件目录

# 先写 .gitignore 避免依赖文件和打包后文件被提交
